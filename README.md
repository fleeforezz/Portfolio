# To test the project
### Requirement :
- NodeJS
- Npm

### Run the project
1. Clone the project
```
git clone https://github.com/fleeforezz/Portfolio.git
``` 
2. Get inside the project
```
cd ./Portfolio/Portfolio
```
3. Install npm 
```
npm install
```
4. Run npm in dev mode
```
npm run dev
```
